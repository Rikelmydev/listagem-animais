# Pet App - Visualização de Pets

## Sobre o Projeto
Este projeto consiste no desenvolvimento de um aplicativo mobile para visualização de pets, criado no Android Studio utilizando a linguagem Java.

A aplicação permite ao usuário navegar por uma lista de pets e, ao selecionar um item, visualizar informações detalhadas sobre o animal.

## Funcionalidades
- Listagem de pets disponíveis  
- Interação por clique em cada item  
- Exibição de informações detalhadas:
  - Nome  
  - Raça  
  - Data de nascimento  

## Objetivo
O objetivo do projeto é aplicar conceitos de desenvolvimento mobile, incluindo criação de interfaces, navegação entre telas e manipulação de dados, proporcionando uma experiência simples e funcional ao usuário.

## Tecnologias Utilizadas
- Java  
- Android Studio  
- XML (layouts Android)  

## Estrutura do Projeto
- MainActivity: responsável pela listagem dos pets  
- Activity de detalhes: responsável por exibir as informações do pet selecionado  
- Layouts em XML para construção da interface
